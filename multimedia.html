• img: Es utilizada para insertar una imagen en una página web
    Sus atributos son:
    • src: Se especifica la URL de la ruta de la imagen W3Schools(2020)
    • alt: Muestra texto alternativo de una imagen al dejar el puntero sobre ella
    • Ancho y alto: Estas propiedades pueden ser especificadas de 2 maneras sus medidas estan en pixeles
      • Style: Se especifica el width y el height entre comillas con los valores correspondientes. Ejemplo: style="width:500px;height:600px;"

      • Width: Se especifica el ancho de la imagen. Ejemplo: width="500"
      • Height: Se especifica el alto de la imagen. Ejemplo: height="500"

  • video: Es utilizada para insertar una imagen en una página web
    Sus atributos son:
    • src: Se especifica la URL de la ruta del video
    • poster: Propiedad que muestra una imagen mientras se descarga el video.
    • preload: Propiedad para habilitar si un video se cargara automaticamente al cargar la página.
    • mediagroup: Establece la sincronización de la reproducción del grupo de medios con este atributo.
    • autoplay: Propiedad que establece si un video se debe reproducir automaticamente.
    • loop: Establece si un video debe ser reproducido cada que termine.
    • muted: Establece si un video debe ser reproducido con sonido.
    • controls: Propiedad que muestra los controles de reproducción.
    • width: Propiedad que indica el ancho del video.
    • height: Propiedad que indica el alto del video.

  • embed: Establece un contenedor para insertar un elemento externo como una imagen o una página web
    Sus atributos son:
    • width: Propiedad que indica el ancho del contendor.
    • height: Propiedad que indica el alto del contendor.
    • src: Especifica la URL donde se encuentra el elemento.
    • type: Especifica el tipo de elemento que va a ser insertado.

  • source: Etiqueta donde se especifica 2 fuentes diferentes del mismo archivo donde el navegador elegira el archivo compatible
    Sus atributos son:
    • media: Se especifican los atributos de estilo.
    • size: Especifica el tamaño de una imagen .
    • src: Especifica la URL donde se encuentra el archivo.
    • srcset: Especifica la URL de una imagen para que se adaptable.
    • type: Especifica el tipo de archivo.

  • object: Es utilizado para insertar una imagen o video. Es el ideal para insertar un archivo tipo Flash. Esta etiqueta acepta los mimso atributos de HTML5
